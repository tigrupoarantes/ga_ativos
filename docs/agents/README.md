# Catálogo de Agentes

Esta pasta guarda prompts operacionais para os multiagentes de desenvolvimento do projeto.

Cada agente foi pensado para uma área do repositório e deve ser usado com ownership claro.

Arquivos:

- `planner-produto.md`
- `frontend-ui.md`
- `frontend-dados.md`
- `backend-edge.md`
- `database-migrations.md`
- `qa-review.md`
- `security-appsec.md`

Uso recomendado:

1. Escolher a feature.
2. Definir quais agentes participam.
3. Dar ownership explícito para cada um.
4. Integrar somente depois dos handoffs.
